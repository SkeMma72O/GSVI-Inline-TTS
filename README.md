# GSVI Inline TTS Extension for SillyTavern

这是一个为 SillyTavern 设计的内联 TTS 扩展插件，专门对接 **GPT-SoVITS (GSVI)**。它能够自动检测聊天消息中的 `<tts>` 标签，生成对应的音频，并在消息下方渲染交互式的播放控制面板。

## 核心功能介绍

### 1. 内联音频播放 (Inline Playback)
- **标签检测**：自动解析消息中的 `<tts char="..." emotion="..." lang="...">文本</tts>` 标签。
- **即时播放**：在对话行下方直接渲染播放按钮，点击即可收听，无需等待全文生成。
- **情感支持**：支持为同一角色指定不同的情感标签（取决于模型能力）。

### 2. 音频缓存与管理 (Caching & Management)
- **双重缓存**：音频数据同时保存在内存和浏览器的 IndexedDB 中。页面刷新或重新打开后，已生成的音频无需重新请求，秒开播放。
- **重生成功能**：如果对某一句生成的语音不满意，点击“重新生成”按钮可立即清理旧缓存并调用 API 重新制作。
- **音频下载**：提供下载按钮，方便将满意的语音保存为本地 `.wav` 文件。
- **服务器保存**：可选功能，将生成的音频上传并保存到 SillyTavern 服务器。

### 3. 角色名 - 模型映射 (Character Mapping) 🌟
- **解耦设计**：无需在提示词中生硬地使用模型全名。
- **自定义映射**：在设置中可以将长模型名（如 `李四-v2-1000step`）映射为简短的角色名（如 `李四`）。
- **智能回退**：如果指定的角色名未找到模型，插件会自动回退到设置中定义的默认模型，防止生成中断。

### 4. 强大的提示词管理 (Prompt Manager)
- **多条目管理**：支持保存多段提示词（Prompt），并可通过拖拽调整注入顺序。
- **智能生成器**：根据当前已获取的角色和情感列表，一键生成用于引导 AI 使用 TTS 格式的规范提示词。
- **多位置注入**：支持将提示词注入到D0或指定深度的对话内容中。

### 5. UI 个性化
- **主题色自定义**：可自由调整播放按钮、进度条和交互组件调颜色，完美契合您的 SillyTavern 主题。
- **毛玻璃效果**：设置弹窗采用现代化的毛玻璃（Glassmorphism）设计，美观且易读。

### 6. 云端配置与音频同步 (Cloud Sync) ☁️
- **设置存在服务器**：插件的所有配置（包括角色映射、提示词、自定义颜色等）都保存在 SillyTavern 服务器端。
- **音频元数据绑定**：生成的音频路径会自动绑定到聊天消息的 `metadata` 中。这意味着在电脑上生成过的对话，在手机端打开时会**自动关联**已有的音频文件，无需重新生成，实现真正意义上的跨设备无缝续玩。（注意：此功能需要开启设置中的 “保存音频到服务器”）
- **跨设备免配置**：无论您是从 PC、手机还是平板登录同一个酒馆，所有设置和已生成的音频都会自动同步。

## 使用说明

### 消息格式
在角色描述或世界书中配置 AI 输出以下格式：
```xml
<tts char="角色名" emotion="情感">对话内容</tts>
```
例如：
- `<tts char="李四" emotion="开心">今天天气真不错！</tts>`
- 如果设置了映射 `李四 -> 李四中文模型`，插件将自动调用后者。

### API 配置
- **端点**：输入您的 GPT-SoVITS API 地址（如 `http://localhost:8001`）。
- **格式**：支持自动检测（Adapter 模式或 GSVI 原生模式）。
